# Contributing to Awesome Gadgets

Thank you for helping grow the Awesome Gadgets list! You can contribute by sharing your gadget or any related resource via a **Pull Request** or an **Issue**.

## Adding a Gadget via Pull Request

1. Fork this repository.
2. Add your entry to the appropriate section in `README.md` following the format:
   ```
   - [gadget-name](https://link-to-repo) - Short description of what the gadget does.
   ```
3. Make sure the link is valid and the description is concise.
4. Submit a Pull Request with a clear title (e.g., `Add my-awesome-gadget`).

### Guidelines

- Descriptions should start with a capital letter and end with a period.
- Check that your entry is not a duplicate.

## Suggesting a Gadget via Issue

If you don't want to open a PR, you can [open an issue](../../issues/new/choose) using the **Share a Gadget** template and we'll add it for you.

## Code of Conduct

Be respectful and constructive. We follow the [Contributor Covenant](https://www.contributor-covenant.org/).
